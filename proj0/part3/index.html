Part 3: The Dolly Zoom — 5 Stills + GIF

Capture five stills at different camera distances while inversely adjusting zoom, then combine them into an animated GIF.

Assignment:

Take five still photos that suggest a dolly-zoom: move the camera forward/backward while inversely zooming to keep the subject roughly the same size. Then, combine these stills into an animated GIF file and include it below.

Stills (5 frames)

Replace the image files below with your own. Keep them in this folder so the paths work.

Animated GIF

After you generate the GIF, save it here as dolly.gif and it will appear below.

How to Make the GIF (offline tools)

Option A — ImageMagick (simple):

# macOS / Windows (recent ImageMagick uses 'magick')
magick -delay 12 -loop 0 still1.jpg still2.jpg still3.jpg still4.jpg still5.jpg dolly.gif

# Linux (often 'convert' is available)
convert -delay 12 -loop 0 still1.jpg still2.jpg still3.jpg still4.jpg still5.jpg dolly.gif

Notes: -delay 12 ≈ 8.3 FPS (100/12). Lower = faster. -loop 0 = loop forever. You can also add -resize 1080x to downscale for smaller files.

Option B — FFmpeg (higher quality with palette):

# 1) Make a palette optimized for your frames
ffmpeg -y -framerate 8 -i still%01d.jpg -vf palettegen=stats_mode=full palette.png

# 2) Create the GIF using the palette
ffmpeg -framerate 8 -i still%01d.jpg -i palette.png -lavfi paletteuse=dither=bayer:bayer_scale=5 -loop 0 dolly.gif

Rename your files to still1.jpg ... still5.jpg so the still%01d.jpg pattern matches (or adjust the pattern).
